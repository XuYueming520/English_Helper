# English Helper

Version: **1.0.0**

English Helper is a lightweight browser project for English-learning tools.
More features will be added over time. Each feature is documented below.

## Features

### 1) Crossword Puzzle Maker

Location: `docs/Crossword Puzzle/`

This feature lets you:

- Add words and clue sentences
- Configure puzzle settings (rows, columns, seed, and generation options)
- Generate crossword layouts directly in the browser
- Toggle answer visibility for learning and review

## Project Structure

- `docs/index.html`: entry page for published tools
- `docs/Crossword Puzzle/`: crossword puzzle maker app (HTML, CSS, JavaScript)
